# Data Analytics

An 4 chapter book on Data Analytics.
- An introduction to Data Analytics
- Univariate / Bivariate / Multivariate Analysis
- AND 6 other relevant chapters but those are the top 2.

the audience is Data Analytics undergraduate students and this is one of the first courses.  We will use a lot of Python and Pandas and Google Colab.  Students should be able to wrangle and feature engineer data so that they are positioned to take AI and Power BI and Tableau classes etc. next

## About This Book

Written by Dr. Ernesto Lee & Professor Carlos Marquez
